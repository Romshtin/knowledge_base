# Knowledge Base

Локальная цифровая база знаний в формате Markdown с семантическим поиском через MCP-сервер.

## Структура

```
knowledge_base/
├── .mcp/                  # MCP-сервер для семантического поиска
│   ├── server.py          # сервер (SentenceTransformers + LanceDB)
│   ├── requirements.txt   # зависимости Python
│   └── README.md          # инструкция по настройке MCP
├── .kb_index/             # векторный индекс (генерируется автоматически)
├── books/                 # исходники книг (локально, не в git)
└── README.md              # этот файл
```

> **Важно:** папка `books/` содержит защищённые авторским правом материалы и исключена из репозитория через `.gitignore`. Она используется только для локального индексирования и личного чтения.

## Быстрый старт

1. Установи зависимости MCP-сервера:
   ```powershell
   cd .mcp
   py -m pip install -r requirements.txt
   ```

2. Подключи MCP-сервер к Claude Code (см. `.mcp/README.md`).

3. Добавляй свои заметки (`.md`) в корень проекта — они автоматически проиндексируются при следующем запуске сервера.

## Технологии

- **LanceDB** — векторная база данных без сервера.
- **SentenceTransformers** (`all-MiniLM-L6-v2`) — эмбеддинги для семантического поиска.
- **MCP (Model Context Protocol)** — интеграция с Claude Code.

## Безопасность

- Никакие секреты, IP-адреса, токены или персональные данные не хранятся в репозитории.
- Защищённые авторским правом материалы исключены из Git через `.gitignore`.
